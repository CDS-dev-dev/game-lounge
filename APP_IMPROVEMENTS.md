# ゲームラウンジ 全体改善ポイント

作成日: 2026-05-27

## 📱 アプリ全体の改善

### 🔴 優先度：高（必須機能・重大な問題）

#### 1. オンライン対戦機能の実装状況確認
**現状**: `/games/geister/online` が存在するがマッチング・対戦機能の動作未確認
**課題**:
- Supabaseとの連携が正常に動作するか
- リアルタイム同期の動作確認
- マッチングロジックのテスト
**改善案**:
- E2Eテストの実装
- 2つのブラウザを使った実際の対戦テスト
- エラーハンドリングの強化

#### 2. ローカル対戦の配置UI未実装
**現状**: ローカル対戦ページも配置UIが未実装の可能性
**改善案**:
- SetupBoardコンポーネントを統合
- Player1とPlayer2の配置画面を分離
- 画面の切り替え演出（「Player2の番です」など）

#### 3. 共通ナビゲーション・ヘッダー
**現状**: 各ページに「戻る」リンクがあるのみ
**改善案**:
- グローバルヘッダーの追加
  - ロゴ/タイトル
  - ゲーム選択メニュー
  - ユーザーメニュー（将来的に）
- パンくずリスト
- ハンバーガーメニュー（モバイル）

#### 4. エラーページ・404ページのカスタマイズ
**現状**: デフォルトのNext.jsエラーページ
**改善案**:
- カスタム404ページ
- エラーページのデザイン統一
- ホームに戻るボタン

#### 5. ローディング状態の統一
**現状**: ページごとにローディング表示がバラバラ
**改善案**:
- 共通Loadingコンポーネント

## 📊 改善の優先順位判断基準

1. **ユーザー体験への影響度**
   - 使いにくい、遊べない → 最優先
   
2. **実装コスト**
   - 低コスト × 高効果 → 優先

3. **完成度**
   - 未完成の機能 → 優先
   - 新機能 → 後回し

4. **技術的負債**
   - バグ・エラー → 最優先
   - リファクタリング → 中優先
