# CPU対戦 E2Eテスト手順書

## テスト環境
- ローカル: http://localhost:3002/games/geister/cpu
- 本番: https://game-lounge-pi.vercel.app/games/geister/cpu

## テストケース

### 1. ページロード
- [ ] ページが正常に表示される
- [ ] 「CPU対戦」タイトルが表示される
- [ ] 「駒の初期配置」セクションが表示される
- [ ] 「配置完了（仮）」ボタンが表示される

### 2. 配置フェーズ
- [ ] 「配置完了（仮）」ボタンをクリック
- [ ] エラーメッセージが表示されない
- [ ] ゲームボードが表示される

### 3. ゲームプレイ
- [ ] 自分の駒（👻または😈）が表示される
- [ ] 相手の駒が👤として表示される（種類が隠されている）
- [ ] 「現在のターン：あなた」が表示される
- [ ] 自分の駒をクリックできる
- [ ] 選択した駒が強調表示される（ring-4 ring-indigo-500）
- [ ] 移動可能なマスが緑色で表示される（bg-green-200）
- [ ] 移動可能なマスをクリックすると駒が移動する
- [ ] 「CPUが思考中...」が表示される
- [ ] CPUが自動で駒を移動する
- [ ] ターンが「あなた」に戻る

### 4. 勝敗判定
以下のいずれかで勝負が決まることを確認：
- [ ] 青いお化け👻の脱出成功
- [ ] 相手の青いお化け👻を全て捕獲
- [ ] 自分の赤い悪魔😈を全て取らせた

### 5. ゲーム終了
- [ ] 勝敗結果が表示される
- [ ] 「もう一度プレイ」ボタンが表示される
- [ ] 「もう一度プレイ」をクリックすると配置画面に戻る

## 既知の問題
